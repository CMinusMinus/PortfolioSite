Word Game

Concept: The rules of this game are simple. You must provide a word that starts with the last letter of the word the player before you provides. For example, if the previous player uses the word "apple," the next player must use a word starting with 'e.' If a word you submit has already been used, you are out. What works: Right now, you can create a game, and add words. The words will be checked for validity on the server-side but it does not remove you from the game.

Languages and Frameworks: Javascript, Node.js/Express.js, Mongodb, AngularJS, HTML/CSS, Bootstrap 3.0. Check it out here.
